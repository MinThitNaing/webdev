Maternity Package

❤️ OG နှင့်ခွဲစိတ်၍ကလေးမွေးဖွားခြင်း ဝန်ဆောင်များပါဝင်သော "Maternity Package "ကိုရွေးချယ်မည်ဆိုပါက 591,000 ကျပ်သာ ကျသင့်မှာဖြစ်ပါတယ်။(50% Discount)






☑️ Package တွင်ပါဝင်သောကုန်ကျစရိတ်များ

  • ✅ ကျယ်ဝန်း၍အဆင့်မြင့်သောပိုးသတ်မှုများသေချာဆောင်ရွက်ထားသည့်ခွဲစိတ်ခန်း အတွက် ကုန်ကျစရိတ်များ
  • ✅ ခွဲစိတ်ရာတွင်အသုံးပြုသည့်ဆေးဝါး၊ချုပ်ကြိုးများအစရှိသည့် ကုန်ကျစရိတ်များ
  • ✅ ကျယ်ဝန်းပြီးပစ္စည်းအပြည့်အစုံဖြင့် အဆင်မြင့် လူနာခန်းအသုံးပြုသည့် ကုန်ကျစရိတ်များ
  • ✅ မွေးကင်းစကလေးအတွက် ပြုစုစောင့်ရှောက်ခြင်းနှင့် အသည်းရောင်အသားဝါ ဘီပိုး ကာကွယ်ဆေး ထိုးခြင်း ကုန်ကျစရိတ်များ
  • ✅ အတွင်းလူနာအဖြစ်တက်ရောက်နေစဉ် ပုံမှန်အသုံးပြုသည့်ဆေးဝါးများ(Medicines)နှင့်ဓါတ်ခွဲစမ်းသပ်စစ်ဆေးမှုများ(Laboratory tests)အတွက် ကုန်ကျစရိတ်များ
  • ✅ နေ့စဉ်တာဝန်ကျ ဆရာဝန် ၊ သူနာပြု ၊ သူနာပြုအကူများ၏ ကျန်းမာရေးစောင့်ရှောက်မှုများ
  • ✅ အမှတ်တရမိသားစုဓါတ်ပုံ လက်ဆောင်

❌Package တွင် မပါဝင်သော ကုန်ကျစရိတ်များ

  • ✅ ခွဲစိတ်ခန်းတွင်ခွဲစိတ်ပေးသော သားဖွားမီးယပ်ဆရာဝန်ကြီး၊ လက်ထောက်ဆရာဝန်နှင့် မေ့ဆေးဆရာဝန်ကြီး ၊ကလေးအထူးကု ဆရာဝန်ကြီး များ၏ ဝန်ဆောင်မှုအတွက်ကုန်ကျငွေ (Surgical team fees) နှင့် အထူးကု ဆရာဝန်ကြီးများ၏ အတွင်း လူနာခန်းအတွင်းလာရောက်လှည့်လည်ကြည့်ရှု ပေးသော ကျန်းမာရေး စောင့်ရှောက်မှု အတွက် ကုန်ကျငွေ (Specialist Round Fees)
  • ✅ အတွင်းလူနာခန်း ( သုံးညအိပ်လေးရက်) ထက်ကျော်လွန်သော အခန်းခနှင့်ကုန်ကျစရိတ်များ
  • ✅ အတွင်းလူနာအဖြစ်တက်ရောက်နေစဉ်အတွင်း သာမန်အခြေအနေ ထက်သာလွန်၍ အသုံးပြုရသော ဆေးဖိုးဝါးခများ ၊ ဓါတ်ခွဲခန်း စစ်ဆေးမှုနှင့်ဓါတ်မှန်ရိုက်ခြင်း အစရှိသည်များ အတွက်ကုန်ကျ စရိတ်များ ( ပုံမှန်အသုံးပြု ထားသော ဆေးဖိုးဝါးခများ အတွက် ပါဝင်တွက်ချက်ပြီးဖြစ်ပါသည်)
  • ✅ လူနာနှင့် ကလေးအတွက် နောက်ဆက်တွဲ ရောဂါဝေဒနာများ ကုသခြင်းနှင့် မိခင်၏မူလရှိရင်းစွဲရောဂါများ ကုသခြင်းအတွက် ကုန်ကျစရိတ်များ
  • ✅ သွေးနှင့် သွေးရည်ကြည်အစရှိသည်တို့ အသုံးပြုရခြင်းအတွက် ကုန်ကျစရိတ်များ
  • ✅ Special Nurse ၊Special Nurse Aid များအတွက် ကုန်ကျစရိတ်များ

  • ✳️ Packageအား အခန်းအမျိုးအစား၊ သက်ဆိုင်ရာဆရာဝန်ကြီးများအလိုက် ရွေးချယ်လိုပါက စိတ်တိုင်းကျဆောင်ရွက်ပေးပါတယ်။
  • ✳️ ကိုယ်ဝန်မအပ်နှံရသေးသည့် မိခင်လောင်းများလည်း စုံစမ်းမေးမြန်းနိုင်ပါတယ်။
  • ✳️ အသေးစိတ်အချက်အလက်များ စုံစမ်းမေးမြန်းမှုကိုဖြေကြားပေးဖို့ ချမ်းမြေ့မေတ္တာ အထွေထွေရောဂါကုဆေးရုံမှ အသင့်ရှိပါတယ်။
  • ✳️ သားဖွားမီးယပ်ဆိုင်ရာ အထူးကုဆရာဝန်ကြီးများက စမ်းသပ်ကုသမှုများကို ၂၄ နာရီ ဝန်ဆောင်မှုပေးလျက်ရှိပါတယ်။

The Whole Journey Maternity Package

❤️ ချမ်းမြေ့မေတ္တာ အထွေထွေရောဂါကုဆေးရုံ မှ မြန်မာနှစ်သစ်ကူး အထိမ်းအမှတ်အဖြစ်ခွဲစိတ်၍ ကလေးမွေးဖွားခြင်း အတွက် ကိုယ်ဝန်ဆောင်စဉ် စစ်ဆေးစမ်းသပ်မှုများမှစတင်၍ ခွဲစိတ်မွေးဖွားပြီး သည်အထိ ဝန်ဆောင်မှုများ ပါဝင် သော “The Whole Journey Maternity Package” အား မိတ်ဆက်ပေးလိုက်ပါတယ်ရှင်။

“The Whole Journey Maternity Package” ကိုရွေးချယ်မယ်ဆိုရင် 713,000 ကျပ်သာ ကုန်ကျမှာဖြစ်ပါတယ်ရှင်။


☑️ Package တွင် ပါဝင်သော ကုန်ကျစရိတ်များ

  • ✅ ကိုယ်ဝန်အတွက်စတင်စစ်ဆေးရသည့်ဓါတ်ခွဲစစ်ဆေးမှုများအပါအဝင် USG(အာထရာဆောင်း)စမ်းသပ်မှု နှစ်ကြိမ်အတွက်ကုန်ကျစရိတ်များ
  • ✅ ကျယ်ဝန်းအဆင့်မြင့်ပြီး ပိုးသတ်မှုသေချာဆောင်ရွက်ထားသော ခွဲစိတ်ခန်း အတွက် ကုန်ကျစရိတ်များ
  • ✅ ခွဲစိတ်ရာတွင်အသုံးပြုသည့်ဆေးဝါး၊ချုပ်ကြိုးများအစရှိသည့် ကုန်ကျစရိတ်များ
  • ✅ ကျယ်ဝန်းပြီး ပစ္စည်းအပြည့်အစုံဖြင့် အဆင်မြင့် လူနာခန်းအသုံးပြုသည့် ကုန်ကျစရိတ်များ
  • ✅ ‌မွေးကင်းစကလေးအတွက် ပြုစုစောင့်ရှောက်ခြင်းနှင့် အသည်းရောင်အသားဝါ ဘီပိုး ကာကွယ်ဆေး ထိုးခြင်း ကုန်ကျစရိတ်များ
  • ‌‌✅ မွေးကင်းစကလေးအတွက် အသုံးပြုသည့် အနှီး ၊ စောင် ၊ အကျင်္ ီအစရှိသည့် အသုံးအဆောင်များ အတွက် ကုန်ကျစရိတ်များ
  • ✅ အတွင်းလူနာအဖြစ်တက်ရောက်နေစဉ် ပုံမှန်အသုံးပြုသည့်ဆေးဝါးများနှင့်ဓါတ်ခွဲစမ်းသပ်စစ်ဆေးမှုများအတွက် ကုန်ကျစရိတ်များ
  • ‌✅ နေ့စဉ်တာဝန်ကျ ဆရာဝန် ၊ သူနာပြု ၊ သူနာပြုအကူများ၏ ကျန်းမာရေးစောင့်ရှောက်မှုများ
  • ✅ အမှတ်တရမိသားစုဓါတ်ပုံ လက်ဆောင်

❌ Package တွင် မပါဝင်သော ကုန်ကျစရိတ်များ

  • ✅ ကိုယ်ဝန်ဆောင်စဉ်အတွင်း သားဖွားမီးယပ်အထူးကုဆရာဝန်ကြီးများ နှင့်ဆွေးနွေးတိုင်ပင်ပြသသည့် ကုန်ကျစရိတ် ( Antenatal Visit)
  • ✅ ခွဲစိတ်ခန်းတွင်ခွဲစိတ်ပေးသော သားဖွားမီးယပ်ဆရာဝန်ကြီး၊ လက်ထောက်ဆရာဝန်နှင့် မေ့ဆေးဆရာဝန်ကြီး ၊ကလေးအထူးကု ဆရာဝန်ကြီး များ၏ ဝန်ဆောင်မှုအတွက်ကုန်ကျငွေ (Surgical team fees) နှင့် အထူးကု ဆရာဝန်ကြီးများ၏ အတွင်း လူနာခန်းအတွင်းလာရောက်လှည့်လည်ကြည့်ရှု ပေးသော ကျန်းမာရေး စောင့်ရှောက်မှု အတွက် ကုန်ကျငွေ (Specialist Round Fees)
  • ✅ အတွင်းလူနာခန်း ( သုံးညအိပ်လေးရက်) ထက်ကျော်လွန်သော အခန်းခနှင့်ကုန်ကျစရိတ်များ
  • ✅ အတွင်းလူနာအဖြစ်တက်ရောက်နေစဉ်အတွင်း သာမန်အခြေအနေ ထက်သာလွန်၍ အသုံးပြုရသော ဆေးဖိုး ၀ါးခများ ၊ ဓါတ်ခွဲခန်း စစ်ဆေးမှုနှင့်ဓါတ်မှန်ရိုက်ခြင်း အစရှိသည်များ အတွက်ကုန်ကျ စရိတ်များ ( ပုံမှန်အသုံးပြု ထားသော ဆေးဖိုးဝါးခများ အတွက် ပါဝင်တွက်ချက်ပြီးဖြစ်ပါသည်)
  • ✅ လူနာနှင့် ကလေးအတွက် နောက်ဆက်တွဲ ရောဂါဝေဒနာများ ကုသခြင်းနှင့် မိခင်၏မူလရှိရင်းစွဲရောဂါများ ကုသခြင်းအတွက် ကုန်ကျစရိတ်များ
  • ✅ သွေးနှင့် သွေးရည်ကြည်အစရှိသည်တို့ အသုံးပြုရခြင်းအတွက် ကုန်ကျစရိတ်
  • ✅ Special Nurse ၊Special Nurse Aid များအတွက် ကုန်ကျစရိတ်
  • ✅ နှစ်ကြိမ်အထက် ပို၍ ဆောင်ရွက်ရသော USG (အာထရာဆောင်း) အတွက်ကုန်ကျစရိတ်

  • ✳️ Packageအား အခန်းအမျိုးအစား၊ သက်ဆိုင်ရာဆရာဝန်ကြီးများအလိုက် ရွေးချယ်လိုပါက စိတ်တိုင်းကျဆောင်ရွက်ပေးပါတယ်။
  • ✳️ ကိုယ်ဝန်မအပ်နှံရသေးသည့် မိခင်လောင်းများလည်း စုံစမ်းမေးမြန်းနိုင်ပါတယ်။
  • ✳️ အသေးစိတ်အချက်အလက်များ စုံစမ်းမေးမြန်းမှုကိုဖြေကြားပေးဖို့ ချမ်းမြေ့မေတ္တာ အထွေထွေရောဂါကုဆေးရုံမှ အသင့်ရှိပါတယ်။
  • ✳️ သားဖွားမီးယပ်ဆိုင်ရာ အထူးကုဆရာဝန်ကြီးများက စမ်းသပ်ကုသမှုများကို ၂၄ နာရီ ဝန်ဆောင်မှုပေးလျက်ရှိပါတယ်။
